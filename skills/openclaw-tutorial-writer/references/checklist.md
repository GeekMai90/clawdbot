# OpenClaw 公开教程撰写检查清单（用于自检）

## 结构
- [ ] 标题明确（是什么 + 做什么）
- [ ] 有“本文目标/适用对象”
- [ ] 有“前置条件”
- [ ] 配置步骤是编号/分节、可按顺序执行
- [ ] 有“验证成功”
- [ ] 有“参数汇总”
- [ ] 有“FAQ/排错”
- [ ] 有“回滚/卸载”（至少说明如何禁用）

## AI Agent 友好
- [ ] 每一步都有：目的 / 操作 / 关键参数 / 预期结果
- [ ] 命令、配置、路径都可复制
- [ ] 没有只给结论不讲动作的段落

## 公开发布
- [ ] 默认强制脱敏：邮箱、手机号、token、API Key、appId/apiId/clientId、chatId、webhook、绝对路径
- [ ] 不包含私人信息/内部链接（除非用户明确允许）

## 内容准确
- [ ] 关键参数解释清楚（含默认值）
- [ ] 文中引用的文件/路径与项目结构一致
- [ ] 如果涉及 cron/提醒：说明 schedule、时区、如何停用
